## MESSAGE ME

A very simple chatbox application. Users can sign up, log in and chat with other users.

## APP DIRECT LINK

[App live on Heroku](https://message-me-on-rails.herokuapp.com)

* Postgresql
* bcrypt
* Bootstrap
* HTML5
* CSS3
* Javascript
* jQuery
* AJAX
* Git
* Heroku
